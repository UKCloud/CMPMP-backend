### Environment Variables
1. SESSION_SECRET: This is the key used for signing and encrypting cookies set by application to mantain session state. It will compute the hash.
2. SESSION_HOST: The host name of database connection.
3. SESSION_PORT: The port number of database connection.
4. SESSION_USER: The username of database user.
5. SESSION_PASSWORD: The password for the database user.
6. SESSION_DATABASE: The name of the database.
